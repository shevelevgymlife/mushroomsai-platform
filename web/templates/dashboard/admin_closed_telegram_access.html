Как подключить к боту (кратко)

  1. Создайте закрытый канал или группу / супергруппу в Telegram.
  2. Добавьте основного бота платформы администратором в этот чат/канал.
  3. Для канала и закрытых групп включите заявки на вступление или пригласительные ссылки — бот одобряет заявку, если у пользователя есть активная подписка по правилам тарифа.
  4. Выдайте боту право банить пользователей (и при необходимости одобрять заявки): так бот сможет убирать пользователя при окончании подписки.
  5. Скопируйте ID чата (число вида -100…) и вставьте в поле ниже.
  6. Для теста укажите сначала тестовый канал/группу; после проверки замените ссылки и ID на боевые.
