Ликвидность / биржа

Биржа для пользователей

Выкл — исчезают пункт «Биржа» в меню, страницы /exchange и весь API /api/exchange (и /api/withdraw). Пул и настройки на этой странице доступны всегда, чтобы снова включить сервис.

Где наполнять пул

Прямо на этой странице, блок «Добавить ликвидность» ниже: поле «+ {{ tn }}» увеличивает резерв токена в пуле, «+ бонусы» — резерв бонусов (рублёвый учёт). Кнопка «Отправить» вызывает POST /admin/add-liquidity.

Дополнительно часть выручки от оплаченных подписок автоматически увеличивает bonus_reserve (настраивается через internal_exchange_income_fraction в site_settings). Блок «Автодолив при росте пользователей» при росте числа аккаунтов может добавлять и токен, и бонусы по вашим числам.

Добавить ликвидность (наполнение пула)

POST /admin/add-liquidity — увеличивает резервы пула (маркет-мейкер). Это основное место ручного пополнения.

Автодолив при росте пользователей

При открытии этой страницы считается число «первичных» пользователей. Если оно выросло с прошлого раза и включён флаг — в пул добавляются указанные объёмы.
