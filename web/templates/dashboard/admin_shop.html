🔗 Рефералы и витрина Макси

Не трогает начисление реферальных бонусов. Только то, что видят пользователи в магазине и какие внешние ссылки показываются приглашённым. Реферальное дерево (referred_by) не переписывается: после окончания Макси + льготных дней витрина снова общая для зрителя, без «возврата» к старому режиму магазина.

Эксклюзивный каталог для рефералов

Если включено и реферер — активный продавец маркетплейса с Макси (или в льготном окне после Макси), приглашённый на /shop видит только товары этого продавца, без общих позиций Neurotrops.

Имеет смысл при режиме «Только выбранные id». Пусто = никого.

Сколько дней после смены тарифа с Макси сохраняются «перки» витрины и своя ссылка магазина для реферера. По истечении — новые гости видят стандартные ссылки/витрину; старые привязки дерева не меняются.

Вопросы «где купить» / магазин — бот отвечает одной ссылкой партнёра, без подмешивания шаблонной Neurotrops.

Своя ссылка без префикса Neurotrops: пользователь с флагом продавца маркетплейса и тарифом Макси (или в льготе) может сохранить произвольный URL в партнёрском магазине — обход глобального префикса из раздела «Реферальная программа».

Видимость ссылки приглашённым: как у Старт+ по подписке, плюс для рефереров в эксклюзивном режиме Макси — их ссылка показывается в реферальных материалах при активном Макси/льготе.
